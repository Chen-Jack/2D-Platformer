
# A 2D Platformer created on the Unity engine. <br/>
This was created in during the CUNY Hackathon for Fall 2017. <br/>

Team: <br/>
Jack C. <br/>
Shelly H. <br/>

the ability to interact with the environment using the items in the inventory.
